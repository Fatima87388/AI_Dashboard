# AI Education Dashboard

This project explores the impact of AI (ChatGPT) on learning using meta-analytic data (Deng et al., 2025).

## 🔍 Project Overview

- Analyzed differences across:
  - Educational stages (K–12 vs. university)
  - Subject areas (health, humanities, sciences, social sciences)
  - Application modes (support tool vs. direct learning)

- Used meta-analytic data to examine how AI influences learning outcomes in different contexts.

- Designed an interactive Tableau dashboard to visualize patterns across datasets.

- Findings show that AI’s effectiveness varies depending on context, subject, and usage.
